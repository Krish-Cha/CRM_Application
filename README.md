Django CRM Application Summary
This project is a Customer Relationship Management (CRM) system developed using the Django framework. The application facilitates seamless interaction between administrators and customers by providing separate dashboards tailored to their roles. Key features include:

Frontend: Built using HTML, CSS, JavaScript, and Bootstrap for an interactive and responsive user interface.
Backend: Powered by Python and Django, with SQL for database management.
Authentication: Integrated login authentication system to ensure secure access for users.
Admin Panel: Allows administrators to:
Manage customer details.
Oversee and control CRM activities.
Customer Panel: Provides customers access to:
View and update their information.
Interact with provided services.
Additional Features:
Real-time data management.
Scalability to accommodate business needs.
The project showcases your ability to develop robust, full-stack web applications and implement role-based access controls.
